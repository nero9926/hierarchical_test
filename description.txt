Описание решения:

Программа состоит из 2-х скриптов:
	1.Отдельный скрипт для создания таблицы и заполнения её данными из JSON.
	2.Отдельный скрипт для получения выборки сотрудников по идентификатору сотрудника.

В запросе на получение выборки сотрудников используется 2 рекурсивных CTE:
	1-ый для того, чтобы от сотрудника по его идентификатору дойти до связанного с ним офиса через parentId.
	2-ой для того чтобы от найденного офиса пройтись по всем связанным с ним сотрудникам.

2 скрипта объединены в одно консольное приложение, в котор по командам можно их запускать.
